Git commands:

add : git add .

commit : git commit -m "message"

push : git push origin main

git status : git status

if in case u create a local file (file in vscode or any other editor) u cant push it directly to the github. for that first u need to create a repository in github (in ur profile) and then u need to copy that repository link,

1. git remote add origin "link"

2. git remote -v

3. git branch

4. git branch -m main

also if u want to add and commit it together then u can use this command:

git commit -am "message"

for only add u can do either of the following:

1. git add .

2. git commit -a

for only commit u can do:

git commit -m "message"

for only push u can do:

git push origin main

also if u write git push -u origin main then u dont need to write origin main again and again. -u is used to set the upstream. after writing this u can directly write git push(instead of git push origin main) and it will push the changes to the main branch because after setting upstream -u (means origin or the place where data has sent is setted) so after that just writing git push is enough.

also if u handling multiple files and if u did some changes in one file and u want to push only that file then u can do this:

first set file by : cd file_name

then add and commit and push it.
